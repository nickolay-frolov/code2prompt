# 🚀 code2prompt

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python Version](https://img.shields.io/badge/python-3.12-blue.svg)](https://www.python.org/)

Консольная микро-утилита для автоматической сборки исходного кода из выбранных папок в один структурированный Markdown-файл. Идеально подходит для быстрой передачи контекста проекта в LLM (ChatGPT, Claude, DeepSeek и др.) без лишнего мусора и ручного копипаста.

---

## ✨ Особенности (Features)

* 📦 **Легковесность:** Работает на стандартной библиотеке Python, не раздувает проект лишними зависимостями.
* 📂 **Рекурсивный обход:** Сканирует указанную папку на всю глубину, сохраняя относительные пути к файлам.
* ⚙️ **Гибкая настройка:** Список сканируемых расширений и синтаксис подсветки задаются через внешний JSON-файл.
* 🤖 **AI-Ready:** Генерирует чистый Markdown с правильными блоками кода (копируй и отправляй нейронке).
---

## 🛠 Стек технологий (Tech Stack)

* **Язык:** Python 3.12
* **Инструменты:** PyInstaller (для компиляции в автономный `.exe`)
* **Формат вывода:** Markdown (`.md`)

## 🚀 Быстрый старт (Getting Started)

### Требования
* **Для готовой сборки:** Windows 10 / 11 (установка Python не требуется).
* **Для запуска из исходников:** Python 3.12 или выше.

### Установка и запуск

1. **Использование готового релиза (Рекомендуется):**
   * Перейдите в раздел [Releases](https://github.com/nickolay-frolov/code2prompt/releases).
   * Скачайте архив, распакуйте его в удобное место.
   * Убедитесь, что файл `mapping.json` лежит в той же папке, что и `code2prompt.exe`.

2. **Запуск из исходников:**
   ```bash
   git clone https://github.com/nickolay-frolov/code2prompt.git
   cd code2prompt
   python code2prompt.py <пути_к_папкам>

## 💻 Использование (Usage)
Передайте один или несколько путей к проектам/папкам через пробел в качестве аргументов. Утилита создаст объединенный файл прямо внутри каждой указанной директории под именем ИмяПапки_combined_code.md.

```bash
code2prompt.exe "C:\MyProject\MyDirectory1" "C:\MyProject\MyDirectory2"
```
⚠️ Важно: Если в пути к папке есть пробелы, обязательно оборачивайте этот путь в двойные кавычки.

## Настройка (mapping.json)
Вы можете полностью контролировать, какие типы файлов утилита должна собирать и как их подсвечивать. Для этого отредактируйте конфигурационный файл, лежащий рядом с утилитой:

```json
{
  ".cs": "csharp",
  ".py": "python",
  ".js": "javascript",
  ".ts": "typescript",
  ".go": "go",
  ".json": "json",
  ".html": "html",
  ".css": "css"
}
```
